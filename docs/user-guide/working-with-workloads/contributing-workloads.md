# Sharing Custom Workloads

You can share a custom workload with other Apache Solr Benchmark users by contributing it to the
[solr-benchmark-workloads](https://github.com/janhoy/solr-benchmark-workloads) repository on
GitHub — the default workload source for Apache Solr Benchmark. Workloads in this repository
are available to all users without needing `--workload-path`.

For the full contribution guidelines — required files, README contents, data and licensing
requirements, testing requirements, the pull request and review process, and the backporting
policy — see the repository's
**[CONTRIBUTING.md](https://github.com/janhoy/solr-benchmark-workloads/blob/main/CONTRIBUTING.md)**.
